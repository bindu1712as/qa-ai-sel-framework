# AI-Assisted Selenium Automation Framework

## Overview
This project demonstrates a production-style Selenium automation framework
enhanced with Machine Learning for intelligent test analytics.

## Tech Stack
- Python
- Selenium
- Pytest
- SQL
- Machine Learning (scikit-learn)
- CI/CD (GitHub Actions)

## Key Features
- Selenium UI automation using Page Object Model
- ML-based flaky test detection
- SQL backend validation
- CI-ready architecture

## Target
Designed for AI-driven product companies to showcase modern QA practices.
